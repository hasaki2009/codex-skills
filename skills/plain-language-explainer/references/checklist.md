# Plain-Language Checklist

Use this quick checklist before responding:

- Have I stated the conclusion in one sentence?
- Have I explained why it matters to the user now?
- Did I include one concrete example?
- Did I provide 1-3 next actions?
- Did I remove or explain jargon?
- Is the response still understandable if the user has zero coding background?

If the answer to any item is "no", rewrite before finalizing.
